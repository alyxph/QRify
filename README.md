# Web QR Generator (Vue + Vite + Tailwind)

Project minimal untuk membuat QR code dengan Vue 3, Vite dan TailwindCSS.

Prasyarat:
- Node.js (LTS) dan `npm` terpasang.

Perintah (PowerShell):

```powershell
cd "d:/belajar/web qr generator"
npm install
npm run dev
```

- Setelah `npm run dev`, buka URL yang diberikan (biasanya `http://localhost:5173`).

Catatan:
- Saya menambahkan dependensi `qrcode` untuk menghasilkan QR pada canvas.
- Jika ingin build untuk produksi, jalankan `npm run build`.
